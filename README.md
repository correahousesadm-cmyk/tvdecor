# TV Decor Landing Page

Landing page estática multilíngue (ES, PT-BR, EN) para TV Decor.

## Deploy no Vercel

1. Importe este repositório no Vercel.
2. Framework Preset: `Other`.
3. Build Command: deixe vazio.
4. Output Directory: deixe vazio (raiz do projeto).
5. Deploy.

## Estrutura

- `index.html` pagina principal
- `css/styles.css` estilos
- `js/main.js` interacoes e traducoes
- `assets/` imagens e logo
- `privacidade/` pagina legal
- `termos/` pagina legal
